# Shadow DOM datePicker

Custom datePicker component, base on Shadow DOM.

## Demonstration

<http://www.toobug.net/shadowDatePicker/example.html>

## How to use

```html
<link rel="import" href="./datepicker.html">

<input is="shadow-datepicker"/>
```
